Global Franchise Control Hub (GFCH)

해외사업팀 마스터 가맹(MF) 관리 ERP 시스템 사용자 및 아키텍처 종합 가이드

안내 사항
본 문서는 GFCH 웹 시스템의 모든 기능 명세, 데이터 동기화 메커니즘, 그리고 유지보수 프로세스를 실무 관점에서 기술한 종합 매뉴얼입니다. 본 문서를 웹 브라우저에서 열어 [PDF 다운로드] 버튼을 누르시면 아주 깔끔한 보고서용 PDF로 즉시 저장하실 수 있습니다.

1. 시스템 개요 및 아키텍처

GFCH는 이메일, 엑셀, 메신저 등으로 파편화되어 운영되던 해외 마스터 가맹(Master Franchise) 파트너사와의 주문, 정산, 물류, 그리고 기밀 문서들을 한곳에서 실시간으로 통합 관리하기 위해 탄생한 웹 기반 차세대 ERP 시스템입니다.

시스템 핵심 아키텍처

  • Frontend: Next.js 16 (React 19, TailwindCSS, Shadcn/ui, Recharts)
  • Database: Firebase Firestore (실시간 실물 데이터 베이스)
  • Authentication: Firebase Auth (이메일 기반 로그인 및 역할 주입)
  • File Storage: Google Apps Script (GAS) API → Google Drive (계약서, 송장, 선하증권 실물 보관)

2. 역할 기반 권한 관리 (RBAC)

시스템의 무결성과 보안을 유지하기 위해 사용자는 두 종류의 역할로 엄격하게 분리됩니다.

사용자 역할 해당 대상 접근 가능한 화면 및 기능
HQ Admin (본사 관리자) 한국 컴포즈커피 본사 해외사업팀 및 임직원 전체 화면 접근 가능
- 파트너사 마진 및 공급가 세팅 권한
- 재무 승인 및 원가 관리
- 모든 파트너사의 기밀 계약 문서 업로드 및 관리
- 파트너사 계정 등록
MF Partner (마스터 파트너) 일본, 베트남 등 현지 마스터 가맹 사업자 제한된 화면만 접근 가능
- 본인 파트너사의 주문 신청 및 물류 상태 조회
- 본인 파트너사의 정산 송장 다운로드
- 본사에서 지정한 최종 Partner 공급가 조회 가능 (원가/본사마진 비공개)

3. 핵심 모듈별 사용 가이드

① 상품 마스터 및 가격 설정 (Products & Pricing)

본사는 원가 변화에 맞추어 실시간으로 파트너별 최종 공급가를 동적으로 제어할 수 있습니다.

실공급가 자동 계산 공식
최종 공급가(Partner Sales Price) = HQ True Cost(실제 본사원가) / (1 - Target Margin Rate %)
  1. 상품 등록/수정 (/orders/products): 상품명, 실제 본사 원가(HQ Cost), 카테고리, 활성화 여부를 설정합니다.
  2. 파트너별 마진 설정 (/orders/pricing): 특정 파트너사를 선택하고, 해당 파트너사에게 적용할 Target Margin %를 입력합니다. 시스템이 원가 대비 최종 Partner Sales Price를 실시간으로 자동 계산하여 테이블에 매핑합니다.

② 통합 주문 보드 (Integrated Order Board - /orders)

주문 접수부터 현지 배송 및 정산 완료까지 총 8단계의 엄격한 생애주기(Lifecycle)를 추적합니다.

임시저장 → 본사승인 → 생산요청완료 → 제품생산완료 → 운송시작 → 항구도착 → 파트너인도 → 정산완료

  • 주문 생성: 파트너가 필요한 품목과 수량을 입력하여 주문을 생성합니다.
  • 증빙 파일 업로드 (B/L, 송장 등): 본사는 물류 이동 단계에 맞춰 선하증권(B/L), 상업송장(C/I) 등의 문서를 주문 건별로 첨부할 수 있습니다. 해당 파일들은 구글 드라이브에 자동으로 안전하게 업로드됩니다.

③ 재무 정산 및 매출 대시보드 (/orders/finance, /sales)

주문별 매출, 실원가, 본사 실마진액을 자동 집계하여 차트와 그리드로 시각화합니다.

  • P&L 대시보드 (/sales): 당월 기준 파트너사별 매출 비중을 원형 차트로 시각화하며, 1월부터 12월까지 연간 누적 매출 추이를 바(Bar) 차트로 한눈에 비교 분석할 수 있습니다.
  • 재무 승인 보드 (/orders/finance): 당월 정산 내역 검토 후 "정산 확정(Lock)" 처리를 통해 데이터 변조를 차단하고, PDF 송장 출력 기능(jsPDF)을 통해 원클릭으로 외화 청구서를 생성할 수 있습니다.

④ 글로벌 계약 및 문서 관리 (Document Hub - /documents)

마스터 가맹 계약서, 비밀유지약정서(NDA), 갱신 이력 및 주요 기밀 PDF 파일들을 클라우드에 백업하고 타임라인을 관리합니다.

  • 문서 등록: 파트너사, 문서 유형(기본/갱신/NDA/기타), 계약 효력 시작일 및 만료일을 기입하고 PDF 실물 파일을 첨부합니다.
  • 만료 상태 모니터링: 계약 만료일이 지난 문서에는 [⚠️ 기간 만료] 배지가 자동으로 활성화되어 갱신 시점을 직관적으로 알려줍니다.

4. 배포 및 유지보수 가이드

본 프로젝트는 GitHub 저장소와 Vercel Cloud Platform이 자동 CI/CD 파이프라인으로 연결되어 개발자가 코드를 Push할 때마다 100% 자동으로 빌드와 무중단 배포가 수행됩니다.

  • 로컬 개발 서버 명령어: npm run dev (Webpack 엔진 탑재)
  • 실서비스(Production) URL: https://gfch-web.vercel.app
